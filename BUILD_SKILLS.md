# ArcAllowance Build Skills

## Product Strategy Skill

Frame ArcAllowance around a precise pain: autonomous AI agents need spend controls before they can safely hold USDC budgets. Keep the demo focused on budgets, allowlists, approvals, and receipts.

## UI/UX Skill

Design a serious dark fintech SaaS interface with clear hierarchy, compact cards, status badges, and demo-ready flows. Prioritize dashboard clarity, simulator usability, and receipt readability.

## Frontend Engineering Skill

Use Next.js App Router, TypeScript, Tailwind CSS, reusable components, and browser-local state. Keep routes stable and client-only state isolated from server-rendered layout.

## Mock Web3 Integration Skill

Model Gateway/x402, Arc transaction memos, Arc transaction hashes, and batch IDs as clearly labeled mock artifacts. Never imply real funds moved.

## Policy Engine Skill

Evaluate merchant allowlists, per-transaction limits, daily limits, purpose controls, high-risk merchants, and human approval thresholds. Return an auditable ordered list of checks.

## DevOps/VPS Deployment Skill

Prepare the app for Ubuntu VPS deployment with Node.js 20+, npm, PM2, Nginx reverse proxy, DNS A records for `arcallowance.xyz`, and Certbot HTTPS.

## Demo/README Writing Skill

Write clear product copy, demo steps, limitations, and future Arc-native integration notes so the project can be shown in a video or posted to the Arc community.
